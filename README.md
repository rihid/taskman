## Installation
### Clone the Repository
Clone the Taskman repository from GitHub to your local machine:

```bash
git clone https://github.com/rihid/taskman
cd taskman
```

### Install Dependencies

```bash
# Using npm
npm install
```
### Start Development Server

```bash
npm run dev
```
